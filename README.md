# "{ - Zocrop.io

Zocrop.io adalah alat pengedit gambar berbasis browser yang sepenuhnya offline dan privat. Didukung oleh teknologi client-side, Zocrop.io memungkinkan pengguna untuk melakukan cropping, zoom, dan penyesuaian gambar secara cepat tanpa perlu mengunggah data ke server.

---

## Fitur Utama

- **Privasi Penuh**: Tidak ada data gambar yang dikirim ke server. Semua proses berlangsung di browser Anda.
- **Zoom Presisi**: Hingga 800% dengan scroll wheel dan pan/drag.
- **Cropping Akurat**: Preset ukuran untuk Instagram, YouTube, TikTok, Open Graph, dan lainnya.
- **Tools Penyelarasan**: Grid dan titik snap untuk komposisi visual yang seimbang.
- **Latar Belakang Kustom**: Warna solid atau transparan.
- **Format Ekspor Beragam**: PNG (lossless), JPEG, WebP, BMP, ICO dengan pengaturan kualitas.
- **Open Source**: Kode lengkap tersedia di GitHub di bawah lisensi MIT.
- **Tanpa Registrasi & Instalasi**: Langsung bisa digunakan di browser secara gratis.

---

## Demo & Penggunaan

1. **Masukkan Gambar**  
   Drag & drop, paste (Ctrl+V), atau klik untuk memilih file.
   
2. **Atur & Edit**  
   Gunakan scroll wheel untuk zoom, drag untuk geser posisi gambar, dan pilih preset ukuran dari sidebar. Aktifkan grid untuk komposisi yang lebih seimbang.
   
3. **Export**  
   Pilih format, atur kualitas, lalu download gambar langsung dari browser.

## Instalasi & Pengembangan

Zocrop.io dibangun menggunakan teknologi berikut:
- HTML5 Canvas
- Vanilla JavaScript
- CSS3 & Tailwind CSS
- Tanpa dependensi eksternal

Anda dapat menjalankan kode sumber secara lokal:

```bash
git clone https://github.com/34labs/zoCrop.git
cd Zocrop.io
npm install
npm run dev
```

Lalu buka browser di `http://localhost:3000` untuk menjalankan versi pengembangan.
